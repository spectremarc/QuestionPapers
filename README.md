# QuestionPapers

Question JSON files used by Eyebell Labs mock tests.

## Papers

- Banking papers use 100 questions.
- JEE Main papers use 75 questions.
- JEE Advanced papers use 102 questions.
- NEET UG papers use 180 questions.

Each exam has three difficulty sets:

- Set 1: Easy
- Set 2: Medium
- Set 3: Hard

The app reads these files through raw GitHub URLs before starting a test.
