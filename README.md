# QuestionPapers

Question JSON files used by Eyebell Labs mock tests.

## Papers

- `papers/banking/ibps-po-ai.json` - IBPS PO AI generated mock
- `papers/jee/main-ai.json` - JEE Main AI generated mock
- `papers/neet/ug-ai.json` - NEET UG AI generated mock

The app reads these files through raw GitHub URLs before starting a test.
