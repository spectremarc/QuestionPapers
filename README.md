# QuestionPapers

Question JSON files used by Eyebell Labs mock tests.

## Papers

- `papers/banking/ibps-po-ai.json` - IBPS PO AI generated mock, 100 questions
- `papers/banking/sbi-po-ai.json` - SBI PO AI generated mock, 100 questions
- `papers/jee/main-ai.json` - JEE Main AI generated mock, 75 questions
- `papers/jee/advanced-ai.json` - JEE Advanced AI generated mock, 102 questions
- `papers/neet/ug-ai.json` - NEET UG AI generated mock, 180 questions

The app reads these files through raw GitHub URLs before starting a test.
